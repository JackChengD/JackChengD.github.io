# 合并区间

# 难度

中等

# 描述

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

# 示例

示例 1：

输入：intervals = [[1,3],[2,6],[8,10],[15,18]] 输出：[[1,6],[8,10],[15,18]] 解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

示例 2：

输入：intervals = [[1,4],[4,5]] 输出：[[1,5]] 解释：区间 [1,4] 和 [4,5] 可被视为重叠区间。

# 思路

先对区间排序，遍历区间，start取后面一个，end取前一个，如果start大于等于end说明有交集可以合并。取区间的最大最小值新区间，然后把之前的去掉，对索引做--，新区间也要进行比较。

# 实现

    var merge = function (intervals) {
        intervals.sort((a, b) => a[0] - b[0]);
        for (let i = 0; i < intervals.length -1; i++) {
            let start = intervals[i][1];
            let end = intervals[i + 1][0];
            if (start >= end) {
                let min = Math.min(...intervals[i], ...intervals[i + 1]);
                let max = Math.max(...intervals[i], ...intervals[i + 1]);
                const cur = [min, max];
                intervals.splice(i, 2, cur);
                i--;
            }
        }
        return intervals;

    };
