# 反转链表

# 难度

简单

# 描述

给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

# 示例

1->2->3->4->5
=>
5->4->3->2->1

示例 1：

输入：head = [1,2,3,4,5] 输出：[5,4,3,2,1]

1->2 =>
2->1

示例 2：

输入：head = [1,2] 输出：[2,1]

示例 3：

输入：head = [] 输出：[]

# 思路

链表最后指向的是null，所以prev开始是null，cur保存当前的head，然后开始while循环，先保存cur的next，因为之后cur的next要指向prev，然后将prev指向cur，cur指向next，继续下面的，这里是有保存下一个节点和断链的过程。

# 实现

var reverseList = function(head) {
    let prev = null;
    let cur = head;
    while(cur) {
        let next = cur.next;
        cur.next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
};
