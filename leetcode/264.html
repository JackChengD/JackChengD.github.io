# 丑数 II

# 难度

中等

# 描述

给你一个整数 n ，请你找出并返回第 n 个 丑数 。
丑数 就是只包含质因数 2、3 和/或 5 的正整数。

# 示例

示例 1：

输输入：n = 10 输出：12 解释：[1, 2, 3, 4, 5, 6, 8, 9, 10, 12] 是由前 10 个丑数组成的序列。

示例 2：

输入：n = 1 输出：1 解释：1 通常被视为丑数。

# 思路

用result存储n之前的所有丑数，然后开始遍历n，找到最小的丑数，然后赋值给当前的result[i]，然后判断当前丑数是2、3、5那个的倍数，对应的记录+1。最后返回result[n]即可。

# 实现

    var nthUglyNumber = function (n) {
        let result = new Array(n + 1).fill(0);
        result[1] = 1;
        let count2 = 1;
        let count3 = 1;
        let count5 = 1;
        for (let i = 2; i <= n; i++) {
            const curCount2Value = result[count2] * 2;
            const curCount3Value = result[count3] * 3;
            const curCount5Value = result[count5] * 5;
            const curValue = Math.min(Math.min(curCount2Value, curCount3Value), curCount5Value);
            result[i] = curValue;
            if (curValue === curCount2Value) {
                count2++;
            }
            if (curValue === curCount3Value) {
                count3++;
            }
            if (curValue === curCount5Value) {
                count5++;
            }
        }
        return result[n];
    };
