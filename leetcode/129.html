# 求根节点到叶节点数字之和

# 中等

简单

# 描述

给你二叉树的根节点 root 和一个表示目标和的整数 targetSum ，判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。
叶子节点 是指没有子节点的节点。

# 示例

graph TD
A((1))-->B((2))
A((1))-->C((3))

示例 1：

输入：root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22 输出：true

graph TD
A((1))-->B((2))
A((1))-->C((3))
B((9))-->D((5))
B((9))-->E((1))

示例 2：

输入：root = [1,2,3], targetSum = 5 输出：false

示例 3：

输入：root = [1,2], targetSum = 0 输出：false

# 思路

从根往下，每次都会乘10+当前节点，就能得到到达该节点时的值。
遍历左右子树，为空是return 0，如果左右节点都为空就代表遍历完毕，return sum。
如果没有遍历完毕继续遍历左右节点。

# 实现

var sumNumbers = function (root) {
    return fn(root, 0);
};

function fn(root, preSum) {
    if (!root) return 0;
    const cur = preSum * 10 + root.val;

    if(!root.left && !root.right) {
        return cur;
    } else {
        return fn(root.left, cur) + fn(root.right, cur);
    }
}
