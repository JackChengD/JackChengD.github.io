# 二叉树的前序遍历

# 难度

中等

# 描述

给你二叉树的根节点 root ，返回它节点值的 前序 遍历。

# 示例

示例 1：

输入：root = [1,null,2,3] 输出：[1,3,2]

示例 2：

输入：root = [] 输出：[]

示例 3：

输入：root = [1] 输出：[1]

示例 4：

输入：root = [1,2] 输出：[2,1]

示例 5：

输入：root = [1,null,2] 输出：[1,2]

# 思路

利用递归，用result存储每个节点，根据中左右进行遍历就行。

# 实现

    var preorderTraversal = function(root) {
        var result = [];
        function fn(root) {
            if(!root) return
            result.push(root.val);
            fn(root.left);
            fn(root.right);
        }
        fn(root);
        return result;
    };
