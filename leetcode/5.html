# 最长回文子串

# 难度

中等

# 描述

给你一个字符串 s，找到 s 中最长的回文子串。

# 示例

示例 1：

输入：s = "babad" 输出："bab" 解释："aba" 同样是符合题意的答案。

示例 2：

输入：s = "cbbd" 输出："bb"

示例 3：

输入：s = "a" 输出："a"

示例 4：

输入：s = "ac" 输出："a"

# 思路

使用中心对称，中心的数字可能是1个，也可能是两个，所以循环的时候两者都需要进行，然后开始向两边进行扩散，相同就继续，不同就退出循环，退出时left已经多减了一次，right多加了一次，需要还原一下，然后与当前最大的长度比较进行更新。

# 实现

    var longestPalindrome = function (s) {
        var result = '';
        for (let i = 0; i < s.length; i++) {
            fn(i, i);
            fn(i, i + 1);
        }
        return result;
        function fn(left, right) {
            while (left >= 0 && right <= s.length - 1 && s[left] === s[right]) {
                left--;
                right++;
            }
            if (right - left - 1 > result.length) {
                result = s.slice(left+1, right);
            }
        }
    };
