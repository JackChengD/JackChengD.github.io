# 面试题 02.02. 返回倒数第 k 个节点

# 难度

中等

# 描述

实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。
注意：本题相对原题稍作改动

# 示例

示例 1：

输入： 1->2->3->4->5 和 k = 2 输出： 4

# 思路

采用快慢指针，先动快指针，每次i会++，当i大于等于k时，慢指针就可以动了，因为这是慢指针+fast先动的i就是指针长度，所以fast走完后，慢指针就会到达倒数第几个。最后输出slow的val就可以找到节点。

# 实现

var kthToLast = function (head, k) {
    let slow = head;
    let fast = head;
    let i = 0;
    while (fast) {
        if (i >= k) {
            slow = slow.next;
        }
        fast = fast.next;
        i++;
    }
    return slow.val;
};
