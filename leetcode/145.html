# 二叉树的后序遍历

# 难度

简单

# 描述

给定一个二叉树，返回它的 后序 遍历。

# 示例

示例 1：

输入: [1,null,2,3] 输出: [3,2,1]

# 思路

利用递归，用result存储每个节点，根据左右中进行遍历就行。

# 实现

    var postorderTraversal = function(root) {
        var result = [];
        function fn(root) {
            if(!root) return;
            fn(root.left);
            fn(root.right);
            result.push(root.val);
        };
        fn(root);
        return result;
    };
