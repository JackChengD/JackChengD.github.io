# 路径总和

# 描述

给你二叉树的根节点 root 和一个表示目标和的整数 targetSum ，判断该树中是否存在 根节点到叶子节点 的路径，这条路径上所有节点值相加等于目标和 targetSum 。
叶子节点 是指没有子节点的节点。

# 示例

示例 1：

输入：root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22 输出：true

示例 2：

输入：root = [1,2,3], targetSum = 5 输出：false

示例 3：

输入：root = [1,2], targetSum = 0 输出：false

# 思路

判断节点是否为空，为空就return false；不为空，则拿到当前节点值，然后开始做差，看结果是不是为0，并且左右节点都不存在了，那么说明到叶节点了，可以return true；不是的话，就看左右节点，将做差的结果传下去。

# 实现

var hasPathSum = function (root, targetSum) {
    if(!root) return false;
    const cur = targetSum - root.val;
    if(cur ===0 && !root.left && !root.right) return true;
    return hasPathSum(root.left, cur) || hasPathSum(root.right, cur);
};
