Linux的基本命令

发表于 2020-04-02 z 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

Linux是最知名和最常用的开源操作系统。作为一个操作系统，Linux是一个软件，位于计算机上的所有其他软件的下面，从这些程序接收请求并将这些请求转发到计算机硬件

我们使用术语“Linux”来指代Linux内核，也是通常与Linux内核捆绑在一起的程序，工具和服务，以提供所有必需的组件全功能操 作系统。有些人，特别是自由软件基金会的成员，将此集合称为GNU / Linux，因为包括的许多工具都是GNU组件。但是并不是所有的Linux安装都使用GNU组件作为其操作系统的一部分。例如，Android使用 Linux内核，但对GNU工具依赖性很低。这个介绍文章部分参考于《linux就该这么学》，希望对于大家的理解有帮助

那么我们就知道了，通常我们说的“linux”其实是指linux内核，而linux操作系统其实是GNU/linux（GNU/linux是指 ，linux内核+GNU组织的软件）

常用命令

1.date——显示日期

date

2.cal——显示日历

cal

3.uname——查看系统

uname

4.uname -r——查看版本

uname -r

5.cd——更换目录

切换到根目录：cd /
切换到上一级目录：cd ..
切换到当前用户家目录： cd

6.pwd——显示当前所在目录

pwd

7.mkdir——建立新目录

普通用法：mkdir abc
建多层次目录： mkdir -p a/b/c
mkdir x/a x/b x/c

8.rmdir——删除空的目录

rmdir a

9.rm -rf——删除文件、目录

rm -rf -a

10.ls——显示文件、目录

查看隐藏文件：ls -a

11.cp——复制文件或目录

拷贝文件：cp a.md b.md
拷贝文件夹 cp -r a a-temp

12.rm—–移除文件或目录

rm install.log 普通删除文件
rm -f install.log 强制删除文件
rm -r -f abc 删除文件夹
rm -rf a 删除文件夹 -r和-f两个短参数可以合到一起

13.cat—–显示全部文档内容

cat nginx.conf

14.less——分屏显示，按b查看下一屏

less nginx.conf

15.head——显示前十行

head nginx.conf

16.tail——显示后十行

tail nginx.conf

17.whereis——寻找特定文件

whereis nginx.conf

18.find——寻找特定文件

find / -name ‘nginx.conf’

19.stat——显示文件的元数据

stat a.md

20.vi——编辑器

vi nginx.conf

21.clear——清屏

clear

总结

作为一个入门运维的小前端，简单的梳理并掌握这些linux命令应该就差不多了吧，其他比较不常用的可以再谷歌或百度一波
