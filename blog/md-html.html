# 背景

最近在搭建自己的博客时，想着如何在html显示.md的文件使文章就可以直接用.md的格式来写，这样方便很多

# 引入将.md转成html的库

# markedjs

引入marked开始转html
具体详情可以看官网：https://github.com/markedjs/marked
直接cdn引入:

 <script src="https://cdn.jsdelivr.net/npm/marked/marked.min.js"></script>

安装npm引入

   npm install markedjs --save

使用

   var marked = require("marked");
   marked.setOptions({
      renderer: new marked.Renderer(),
      gfm: true,
      tables: true,
      breaks: false,
      pedantic: false,
      sanitize: false,
      smartLists: true,
      smartypants: false
   });

    document.getElementById('content').innerHTML = marked('# Marked in the browser\n\nRendered by **marked**.');

这时候就可以在html上正常看到.md转成html，可是它却没有.md的样式，这是需要我们自己引入的

# 如果使用webpack打包的项目

需要让js认识.md文件
先引入html-loader、markdown-loader

   npm install html-loader、markdown-loader --save

修改webpack配置，在rules添加以下代码，否则编译通不过

   { 
      test: /\.md$/,
      use: [ 
         { loader: 'html-loader' },
         { loader: 'markdown-loader', options: {}} 
      ]
   },
   {
      test: /\.js$/,
      include: [/marked/],
      use: [
         'babel-loader',
      ]   
   }

# 引入.md的样式

这里采用github-markdown-css，当然你也可以选择你自己喜欢的
github: https://github.com/sindresorhus/ github-markdown-css
安装npm引入

   npm install github-markdown-css --save

使用

   import 'github-markdown-css';

   <div class="markdown-body" id="content"></div>

# 总结

只需要将.md转成html再引相应的样式，最后显示即可
