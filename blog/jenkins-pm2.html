# 背景

承接我上一篇文章(jenkins部署vue项目)，使用pm2部署，其实也就是改个shell命令再安装下pm2

# 环境搭建

# 安装pm2

首先服务器需要安装pm2

npm install pm2 -g

# 构建express

构建和上一篇文章是一样的，只需要修改shell命令即可，可使用以下命令


echo '============check ENV================' 
whoami
which npm
#npm install -g cnpm --registry=https://registry.npm.taobao.org
node -v
npm -v
cnpm -v

echo '============================begin build======================================='
cd $WORKSPACE
#执行命令安装
cnpm install
pm2 start bin/www
echo '============================end build======================================='

这时候基本上就差不多了，你可以看到pm2已经守护了进程www，浏览器也可以访问该url

# 总结

jenkins环境搭建成功后，后面部署就是一件简单的事情了，只需要构建修改下shell命令就差不多了 环境的搭建才是比较累人的，后面就自动化部署就非常香了，代码提交到github上，点一下部署就行，很香
