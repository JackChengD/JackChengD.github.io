# 环境搭建

# git安装

找到Git (opens new window)进行傻瓜式安装 通过cmd查看是否安装成功 git --version，若打印出版本即安装完毕

# node安装

找到Node (opens new window)进行傻瓜式安装 通过cmd查看是否安装成功 node --version，npm --version。若打印出版本即安装完毕

# hexo-cli安装

通过npm install hexo-cli -g，对hexo-cli进行全局安装

# 初始化项目

hexo init blog：初始化项目，耐心等待 cd blog：进入blog文件 npm install：安装依赖

# 文件介绍

node_modules：依赖包 public：存放生成的页面 scaffolds：生成文章的一些模板 source：用来存放你的文章 themes：主题 _config.yml：博客的配置文件

# 启动

hexo clean：清除你之前生成的东西，也可以不加 hexo generate：生成静态文章，可以用hexo g缩写 hexo server 启动hexo，可以在浏览器localhost:4000查看你的博客，可以用hexo s缩写

# 部署到个人GitHub

# github创建个人仓库

在这里默认你已经有自己的github并且密钥都配置好了 这时你可以在Github.com中点击New repository新建仓库，创建一个和你用户名相同的仓库，后面加.github.io，只有这样，将来要部署到GitHub page的时候，才会被识别，也就是xxxx.github.io，其中xxx就是你注册GitHub的用户名

# hexo配置

现在需要我们将hexo和github关联起来。在你初始化的hexo文件找到_config.yml，翻到最后做如下配置 将repository中的JackChengD修改为个人的GitHub账户名

    deploy:
        type: git
        repository: git@github.com:JackChengD/JackChengD.github.io.git
        branch: master

# 最后一步

这时候，你已经基本完成hexo个人博客的搭建，只需要将以下命令操作一下就可以成功了 hexo clean---清除你之前生成的东西，也可以不加 hexo generate---生成静态文章，可以用hexo g缩写 hexo deploy---部署文章，可以用hexo d缩写 操作完毕后，你就可以访问到自己的github博客，地址栏输入jackchengd.github.io。将jackchengd改为你个人GitHub账户就可以访问啦

# 总结

利用hexo搭建个人博客是不是非常的简洁快速，你还可以选择个人主题对你的博客做更多美化哟，但接下来的操作需要你个人手动去实现哦，博主就不多说啦，加油，学习之路永不停止
