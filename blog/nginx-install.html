# 背景

前两天买了个阿里云的服务器，所以尝试了一手，如何安装使用nginx，在这分享给大家

# nginx

# 安装nginx

1.安装gcc gcc-c++(如新环境,未安装请先安装)

    $yum install -y gcc gcc-c++

2.安装nginx

    1. $ cd /usr/local/
    2. $ wget http://nginx.org/download/nginx-1.8.0.tar.gz
    3. $ tar -zxvf nginx-1.8.0.tar.gz
    4. $ cd nginx-1.8.0
    5. $ ./configure --user=nobody --group=nobody --prefix=/usr/local/nginx --with-http_stub_status_module --with-http_gzip_static_module --with-http_realip_module --with-http_sub_module --with-http_ssl_module
    6. $ make && make install

3.启动命令

    启动：$ /usr/local/nginx/sbin/nginx
    重新加载：$ /usr/local/nginx/sbin/nginx –s reload
    停止：$ /usr/local/nginx/sbin/nginx –s stop
    测试配置文件是否正常：$ /usr/local/nginx/sbin/nginx –t
    强制关闭：$ pkill nginx

4.是否成功

通过个人服务器ip查看，默认端口80

# 修改nginx启动命令

  1. 修改以下配置

在 **/etc/init.d/**下创建名为nginx的启动脚本即可，内容如下：

#!/bin/bash
#
# chkconfig: - 85 15
# description: Nginx is a World Wide Web server.
# processname: nginx

nginx=/usr/local/nginx/sbin/nginx
conf=/usr/local/nginx/conf/nginx.conf
case $1 in
start)
echo -n "Starting Nginx"
$nginx -c $conf
echo " done"
;;
stop)
echo -n "Stopping Nginx"
killall -9 nginx
echo " done"
;;
test)
$nginx -t -c $conf
;;
reload)
echo -n "Reloading Nginx"
ps auxww | grep nginx | grep master | awk '{print $2}' | xargs kill -HUP
echo " done"
;;
restart)
$0 stop
$0 start
;;
show)
ps -aux|grep nginx
;;
*)
echo -n "Usage: $0 {start|restart|reload|stop|test|show}"
;;
esac

设置执行权限：chmod +x /etc/init.d/nginx
注册成服务：chkconfig -add nginx
设置开机启动：chkconfig nginx on

  1. 修改后

之后，就可以使用以下命令了
启动：service nginx start
停止：service nginx stop
重新加载：service nginx resload
service nginx reload

# nginx部署vue项目

  1. vue打包

个人通过npm run build生成dist文件

  1. 将文件传到centOS系统，远程服务器可以通过Xshell、Xftp

  2. 开始部署

进入 /usr/local/nginx/conf
开启vi编辑器 vi nginx.conf
添加以下内容

     server { 
         listen  Vue项目的端口号;
         server_name  localhost;
         #charset koi8-r;
         #access_log  logs/host.access.log  main;
         location / {
          root vue的dist文件夹的路径，比如：/root/guobao/dist;
          index index.html index.html;
        }
     }
  1. 补充，开启多端口

除了添加server，配端口以外，还需要，比如：个人阿里云上开启端口，否则也是访问不了新添加的server

  1. 重新加载

每次修改nginx.conf文件后，都要重新加载

   $service nginx reload

# nginx部署nuxt项目

# 安装node环境

  1. 安装wget
    $yum install wget -y
  1. 安装node
    1. 下载： $ wget https://nodejs.org/dist/v10.13.0/node-v10.13.0-linux-x64.tar.xz
    2. 解压： $ xz -d node-v10.13.0-linux-x64.tar.xz
    3 .解压： $ tar xvf node-v10.13.0-linux-x64.tar
    1. 安装nodejs
       1. cd node-v10.13.0-linux-x64
　　　　2. cd bin
　　　　3. mv ./node /bin/
　　　　4. mv ./npm /bin/
　　　　5. mv ./npx /bin/
　　　　6. cd .. cd lib
　　　　7. mv ./node_modules /lib/
    8. 安装nrm：$ npm install nrm -g
    9. 安装pm2：$ npm install pm2 -g
    10. 查看版本：$ node --version
  1. 开始部署
  1. 拷贝文件
    拷贝文件（.nuxt、static、nuxt.config.js、package.json）至服务器部署目录（比如：/root/nuxt）
  2. 测试 cd /root/nuxt
    npm install -production
    npm run start
    发现可以运行
  3. nginx配置
    进入 /usr/local/nginx/conf
    开启vi编辑器 vi nginx.conf
    添加以下内容
 server {
     listen  Nuxt项目的端口号;
     server_name  localhost;
     #charset koi8-r;
     #access_log  logs/host.access.log  main;
     location / {
      root vue的dist文件夹的路径，比如：/root/nuxt;
      # 需要做反向代理
      proxy_pass http://127.0.0.1:3000; 
    }
 }

4. 重新加载  
每次修改nginx.conf文件后，都要重新加载

$ service nginx reload
